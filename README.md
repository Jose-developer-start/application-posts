## Aplication CRUD REACT JS AND LARAVEL

### Instalation Backend
1. required packege
```php
composer install
```

2. install npm

```php
  npm install
```

### execute migration

```php
php artisan migrate --seed
```

2. ejecutar el servidor 

```php
php artisan serve
```
3. Enlazar las images

```php
  php artisan storage:link
```

4. ejecutar el servidor de npm (laravel mix) =>Fron End

```php
  npm run watch
```
